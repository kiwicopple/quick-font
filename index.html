Quick, Font!

Some good fonts for prototyping or snazzing up an internal tool. You can use the fonts without downloading or installing - just load them directly from Github using the instructions provided.

DISCLAIMER
Many of these fonts are commercial - you have to pay if you want to use them beyond just testing them out. I take no responsibility or liability if you use these on a commercial project. I'm likely breaking some rules even with this site, however I hope that the owners of the fonts will see this as free promotion and advertising. If you are an owner of one of these fonts and would like me to remove it, please open an issue here: https://github.com/kiwicopple/quick-fonts/issues

Avenir

Hello World

There are many ways to say hello.

You can say it boldly.

You can say it pretentiously.

Load
<link rel="stylesheet" href="https://raw.githubusercontent.com/kiwicopple/quick-fonts/master/css/avenir.css" />
Use
<style> p { font-family: "Avenir"; } </style>
