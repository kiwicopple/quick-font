Quick Fonts

Some good fonts for prototyping. You can use this without downloading/installing the fonts - just load them directly from Github.

Note that most of them are commercial - you have to pay if you want to use them beyond just testing them out.

Avenir

Hello World

There are many ways to say hello.

You can say it boldly.

You can say it pretentiously.

Load
<link rel="stylesheet" href="https://raw.githubusercontent.com/kiwicopple/quick-fonts/master/css/avenir.css" />
Use
<style> p { font-family: "Avenir"; } </style>
